# 製作背景（意図）
付与されるポイントは全国一律での設定ですが、希望者が購入して使用する食事券は、  
地域により値段にバラつきがあるため、栃木県という限定的な形での作成を行いました。  
他サイトでは計算ページまで辿り着くまでが長く、ログイン機能なども付けずにシンプルに  
することを他との差別化とし作成しました。
